**Title:** Vacation Planner

**Purpose of app:** To provide an easy tool for those who want to plan vacations with no limit on 
vacations or excursion. The app also provides helpful reminders for when their vacation begins and 
ends, if desired.

**Operating Directions:** From the Welcome screen, tap the purple "View Vacations" button. This wil take 
you to the Vacations screen where a list of the current vacations will show. From here, you can tap 
the three dots in the right corner and select the "Add Sample" button to ad some example vacations 
and excursions. You can also tap on the floating plus button in the bottom right corner to create a 
new vacation. This takes you to the vacation details screen with the fields blank. You can tap on 
the Title and Hotel fields to type in the information. For the dates, tap on the current dates and a
widget will sho up for you to select the correct date. Once you are satisfied with the information, 
tap the three dots in the corner and select save. If you choose to edit or simply view the details 
of a vacation, you can tap a vacation and the app will take you back to the Vacation Details screen 
for that activity. There you can make changes and select save again or, from that same dropdown 
menu, delete the vacation, share the vacation details, set a start/end reminder for the vacation. 
From this same screen, you can view the list of attached excursions to the vacation beneath the main
four details. You can add a new excursion by tapping the plus button in the bottom right corner. 
There you can enter the title and the start date and save it the same way that you did for the 
vacation. If you choose to view or edit any excursion, fom its related vacation screen, tap on your 
desired excursion and you will see the excursion detail screen again. There you can change any 
details and save them, delete the excursion, or set a start date reminder in the top right corner.
